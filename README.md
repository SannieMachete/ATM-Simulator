# 🏧 NEXUS ATM Simulation

A web-based ATM simulation that runs in your browser.

---

## What It Does

- Login with a 4-digit PIN
- Deposit and withdraw money
- Check your balance
- View transaction history
- Admin panel to manage all accounts

---

## How to Run

1. Download `atm-simulation.html`
2. Double-click the file
3. It opens in your browser — done
